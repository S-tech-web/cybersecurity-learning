# Day 2 - TCP, UDP and Ports Lab

## Objective

Understand TCP, UDP and common network ports.

## Commands Used

ss -tuln

sudo ss -tulpn

curl https://google.com

ssh localhost

## Observations

* TCP provides reliable communication.
* UDP focuses on speed.
* Ports help identify services on a device.
* HTTPS commonly uses port 443.

## Common Ports Learned

22 - SSH

53 - DNS

80 - HTTP

443 - HTTPS

## Conclusion

TCP and UDP are transport protocols. Ports allow multiple services to operate on the same machine.
