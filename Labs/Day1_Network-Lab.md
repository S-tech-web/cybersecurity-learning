# Day 1 Network Lab

## Objective

Understand basic networking commands and DNS resolution.

## Commands used

* ip addr
* ping google.com
* nslookup google.com

## Observations

* My device has a local IP address assigned.
* Google.com resolves to an IP address using DNS.
* Ping shows response time in milliseconds.

## Conclusion

Networking communication depends on IP addresses, and DNS is required for human-readable domains.
